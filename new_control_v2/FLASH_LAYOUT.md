# Flash 存储布局

## MCU 信息

| 项目 | 值 |
|------|-----|
| 型号 | STM32G474VET6 |
| Flash 总量 | 512 KB |
| Flash 起始地址 | `0x08000000` |
| Flash 结束地址 | `0x0807FFFF` |
| 页大小 | 2 KB（2048 字节） |
| 总页数 | 256（页号 0 ~ 255） |

---

## 整体布局

```
0x08000000 ┌─────────────────────┐
           │                     │
           │   固件代码区          │  ~60 KB
           │   (链接脚本预留 256KB) │
           │                     │
0x0803FFFF ├─────────────────────┤  ← 链接脚本范围结束
           │                     │
           │   未使用             │  ~248 KB
           │                     │
0x0807E000 ├─────────────────────┤  ← Flash 存储区起始 (第 252 页)
           │  Cell 0 标定数据     │  第 252 页 (0x0807E000 ~ 0x0807E7FF)
0x0807E800 ├─────────────────────┤
           │  Cell 1 标定数据     │  第 253 页 (0x0807E800 ~ 0x0807EFFF)
0x0807F000 ├─────────────────────┤
           │  未使用              │  第 254 ~ 255 页
0x08080000 └─────────────────────┘  ← Flash 物理结束
```

## 标定数据结构体 `CalibFlashData_t`

### C 定义 (`calib_mode.h`)

```c
typedef struct
{
    uint32_t     magic;                        // 4 字节，魔数 0x42495345 ("BISE")
    uint8_t      cell;                         // 1 字节，标定的 Cell 编号 (0 或 1)
    uint8_t      reserved[3];                  // 3 字节，对齐填充
    CalibStep_t  step[CALIB_DUTY_COUNT];      // 46 步标定记录 (46 × 16 = 736 字节)
    uint16_t     crc16;                        // 2 字节，CRC16-CCITT 校验
    // 编译器对齐填充：2 字节
} CalibFlashData_t;
// 总大小: 748 字节（含对齐）
```

### 内存布局表

| 偏移 | 大小 | 字段 | 类型 | 说明 |
|------|------|------|------|------|
| 0 | 4 | `magic` | `uint32_t` | 魔数 `0x42495345` ("BISE") |
| 4 | 1 | `cell` | `uint8_t` | Cell 编号（0 或 1） |
| 5 | 3 | `reserved` | `uint8_t[3]` | 对齐填充（写 0） |
| 8 | 736 | `step[46]` | `CalibStep_t[46]` | 46 步标定记录 |
| 744 | 2 | `crc16` | `uint16_t` | CRC16-CCITT 校验值 |
| 746 | 2 | (padding) | — | 编译器对齐填充 |

## 单步记录 `CalibStep_t`

### C 定义

```c
typedef struct
{
    float   duty;          // 4 字节，占空比（正=制冷, 负=加热）
    float   temp_ch0;      // 4 字节，CH0 稳态平均温度 (°C)
    float   temp_ch1;      // 4 字节，CH1 稳态平均温度 (°C)
    uint8_t valid;         // 1 字节，是否有效 (1=稳定达标, 0=超时)
    uint8_t settled;       // 1 字节，是否达到稳态
    uint16_t reserved;     // 2 字节，对齐填充
} CalibStep_t;
// 总大小: 16 字节
```

### 内存布局表

| 偏移 | 大小 | 字段 | 类型 | 说明 |
|------|------|------|------|------|
| 0 | 4 | `duty` | `float` | 占空比，范围 `+0.45` ~ `-0.45` |
| 4 | 4 | `temp_ch0` | `float` | CH0 稳态温度 (°C) |
| 8 | 4 | `temp_ch1` | `float` | CH1 稳态温度 (°C) |
| 12 | 1 | `valid` | `uint8_t` | `1`=稳定达标, `0`=超时 |
| 13 | 1 | `settled` | `uint8_t` | `1`=达到稳态, `0`=未达到 |
| 14 | 2 | `reserved` | `uint16_t` | 对齐填充 |

## 46 步占空比序列

| Step | 占空比 | 效果 |
|:---:|:---:|:---:|
| 0 | **+0.45** | ❄️ 制冷（最大值） |
| 1 | +0.43 | ❄️ 制冷 |
| ... | ... | ... |
| 22 | +0.01 | ❄️ 微冷 |
| 23 | -0.01 | 🔥 微热 |
| ... | ... | ... |
| 44 | -0.43 | 🔥 加热 |
| 45 | **-0.45** | 🔥 加热（最大值） |

步进：`-0.02`（从制冷 → 加热）

## CRC16 校验

| 项目 | 值 |
|------|-----|
| 算法 | CRC16-CCITT（查表法） |
| 初始值 | `0xFFFF` |
| 多项式 | `0x1021` |
| 校验范围 | `magic` 到 `step[45]`（不含 `crc16` 字段本身） |
| 代码位置 | `My/calib_mode.c` → `CalibMode_CRC16()` |
